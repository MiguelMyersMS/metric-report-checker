## Goal
Build a dashboard with key metrics for each PLG priority area, plus placeholders for metrics covered in other features. Supports Growth - Common Measurement Framework.

## Priority Areas & Metrics

| P# | Priority | Measurements of Success |
|----|----------|------------------------|
| P1 | Diversify Acq Channels | New user & new customer signups by acquisition channel; Channel specific activation rate; Rayfin free SKU and consumer email trial signups |
| P2 | Accelerate Time to Value | % of Dev MAU that achieves CVI in month 1; Median time-to-CVI by workload |
| P3 | Improve Pro Dev Experience | Pro Dev NPS (baseline: 9, all-up: 28); Pro Dev MAU; Shell + observability feature adoption |
| P4 | Convert Trials at Readiness | Trial-to-paid conversion rate (baseline: 11%); Conversion rate by acquisition path; Time from bowie threshold to conversion event |
| P5 | Drive Workload Intensity & Expansion | # customers >$100 CU across 3 hero workloads (baseline: 3.2K); Workload-level NRR for DEADS, DI, RTI |
| P6 | Skill the Ecosystem | DP skill + DP 700 certifications (target: 100K); Training event fill rates by program |
| P7 | Migrate Through Partners | Partner-sourced pipeline (target: $100M); Migration completions by source platform |
